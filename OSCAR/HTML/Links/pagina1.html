Links

São utilizados para fazer a ligação entre duas páginas
Nesta página daremos três exemplos de links:

  1. No primeiro link vamos acessar uma página que está na mesma pasta
  2. O segundo link vai fazer a ligação com uma página de outra pasta
  3. No último link vamos acessar uma página que pode estar em qualquer lugar na web

o link é criado com tag "a" composta por:
    • a tag a
    • o endereço da página a ser acessada
    • o texto que será exibida do link na página é o texto

    Exemplos:
    abrir página dois
    ir para dados pessoas
