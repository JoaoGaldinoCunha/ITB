Formatação de links

Você pode utilizar estilo in-line para formatar links.
Entre as formatações básicas temos:

  1. a:link define o link quando não foi selecionado
  2. a:hover define o link quando o mouse está sobre ele
  3. a:visited define o link que já foi selecionado
  4. a:active define o link ao ser selecionado

A formatação é realizada da seguinte forma:
  • a selector
  • :pseudoclasse
  • {formatação}
